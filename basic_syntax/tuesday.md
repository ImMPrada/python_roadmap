¡Buenas, parceros! 🙌 Esta semana le vamos a meter el diente a la **basic syntax** de Python, o sea, las reglas de cómo se escribe el código para que el intérprete lo entienda.

Lo bonito de Python es que está diseñado para leerse casi como inglés. Mira este ejemplo:

```python
nombre = "Mariana"
print(f"¡Hola, {nombre}! Bienvenida a Python")
```

Tres cositas clave para empezar:

- 🔤 Una **variable** se crea simplemente asignándole un valor con `=`. No hay que declarar el tipo.
- 🖨️ `print()` es la **function** que usamos para mostrar cosas en pantalla.
- ✨ Esa `f"..."` se llama **f-string** y nos deja meter variables dentro del texto con `{}`. La vamos a usar full esta semana.

Una característica que hace a Python ÚNICO: no usa llaves `{}` ni `;` para organizar el código. Usa **indentation** (los espacios al inicio de la línea). Pero de eso hablamos mañana 👀

👉 Para ir calentando: instala Python (si no lo tienes) y corre ese `print` de arriba. Cuéntanos en el hilo qué te salió. ¡Vamos con toda! 🚀
