Hoy nos metemos a la práctica con tres piezas que vas a usar TODOS los días en Python. 💪

**1. Variables y tipos básicos**

```python
nombre = "Camilo"      # string (texto)
edad = 25               # int (entero)
estatura = 1.75         # float (decimal)
es_estudiante = True    # bool (verdadero/falso)
```

No declarás el tipo: Python lo deduce solo. Eso se llama *dynamic typing*.

**2. Comentarios**

```python
# Esto es un comentario de una línea: Python lo ignora
print("Hola")  # también puedes comentar al final de una línea
```

**3. f-strings** (nuestra forma favorita de armar texto)

```python
print(f"{nombre} tiene {edad} años y mide {estatura} m")
```
