¡Cerramos la semana, parceros! 🎉 Hagamos un recap rápido de lo que recorrimos sobre **variables** y **data types**:

- 📦 Una **variable** guarda y etiqueta datos en memoria; se crea con `=` y se le pone un nombre descriptivo.
- 🔢 Los tipos básicos: **int**, **float**, **str** y **bool** (y el especial **None** / **NoneType**).
- 🔎 La **function** `type()` te dice qué tipo tiene cualquier valor.
- 🤯 Las variables son **etiquetas que apuntan** a datos, no cajas que los contienen.
- 🔄 Con **casting** (`int()`, `float()`, `str()`) conviertes entre tipos, clave porque `input()` siempre devuelve **str**.

Una cheat sheet para que guardes 📌:

```python
entero  = 7        # int
decimal = 3.14     # float
texto   = "hola"   # str
verdad  = True     # bool
vacio   = None     # NoneType

print(type(texto), int("42") + 8)   # <class 'str'> 50
```
