¡Buenas, parceros! 🙌 Esta semana le vamos a meter el diente a las **variables** y los **data types** de Python. Arrancamos por lo más básico: ¿qué es una **variable**?

Piénsalo así: una **variable** es como una cajita con una etiqueta donde guardas un dato para usarlo después. Le pones un nombre descriptivo y adentro queda la información guardada en memoria. 🧠

```python
nombre = "Mariana"
edad = 23
print(f"{nombre} tiene {edad} años")
```

Tres cositas clave para hoy:

- 🏷️ Para crear una **variable** solo le asignas un valor con `=`. En Python NO tienes que declarar el tipo: él lo deduce solo.
- 📛 El nombre debe ser descriptivo: `precio_total` se entiende mucho mejor que `x`. Tu yo del futuro te lo agradece.
- 🔁 El valor de una **variable** se puede cambiar cuando quieras: por eso se llama *variable*.

¿Por qué importa tanto esto? Porque casi todo programa consiste en **guardar datos**, **transformarlos** y **mostrarlos**. Sin variables no hay nada que mover. 🚀
