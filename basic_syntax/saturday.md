Sábado relax, pero igual sumamos. 😎 Hoy les dejo un **recurso buenísimo** para repasar la basic syntax sin enredarse:

📄 **Learn X in Y Minutes — Python**
https://learnxinyminutes.com/docs/python/

¿Por qué me gusta tanto? Porque es UNA sola página donde ves casi toda la sintaxis del lenguaje a punta de ejemplos comentados. Ideal para tener abierta mientras programas y consultar rapidito.

Un mini-aperitivo de lo que vas a encontrar:

```python
# Varias asignaciones en una línea
a, b, c = 1, 2, 3

# Intercambiar valores sin variable temporal (¡muy pythonico!)
a, b = b, a

# Texto multilínea con triple comilla
mensaje = """Esto ocupa
varias líneas"""
```

Ese truco de `a, b = b, a` para intercambiar **variables** es de las cosas que hacen que la gente se enamore de Python. ✨
